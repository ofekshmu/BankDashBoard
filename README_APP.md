# Bank project

## Summary
An app personaly made to parse and display Bank account data, using the raw data downloaded from the Bank's official site.

### Input files
The app was originally built based on 3 different input files all generated by Bank Leumi official website, downloaded from my personal account. Each file display different data, while there are many intersections and data repetions across files.The files are devided to 3 categories:

1. Outer credit cards transactions

    All transactions made on credit cards that belong to other companies and are visible trough the Banks site.

2. Inner credt cards transactions

    All transactions made on credit cards that belong to bank Leumi.

3. Bank transactions

    Transaction made directly to the bank account, such as money transfers, credit card update at month's end and such.


Files are structured differently and therefore require different parsing, handling, inserting and manipilations.
